Java 的 LinkedList 中 add 方法与 push 方法的区别
发表于2023-03-19|更新于2023-05-04
Java 中没有栈结构，这是因为早期的栈基于 vector 实现，而 vector 是可以随机访问的，破坏了栈的封装，后来 vector 被废弃，stack 也被废弃。 目前可以使用 Java 中的 Deque 双端队列来实现栈的相关操作，而双端队列的常用实现类是 LinkedList 和 ArrayDeque。 在 LinkedList 中，提供了 add 和 push 方法添加元素。 其中 add 方法用于在队尾添加元素，而 push 则在队头添加元素。 相对应的 pop 方法在队头删除元素，因此 push 和 pop 方法相结合才能实现正确的栈操作，不可以和 add 混用。 实际上 LinkList 提供了非常丰富的删除和添加方法，大部分都可以指定在队头或队尾添加，因此尽可能使用没有歧义的方法。
